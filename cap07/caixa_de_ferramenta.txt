CAIXA DE FERRAMENTAS C

Voce ja tem o capitulo 7 no seu repertorio e, agora, acrescentou funcoes
avancadas a caixa de ferramentas.

* Ponteiros para funcoes deixam passar funcoes ai como se fossem dados.
* Ponteiros para funcoes sao os unicos ponteiros que nao precisam dos operadores * e &
* ...mas voce ainda pode usa-los se quiser.
* O nome de cada funcao e um ponteiro para a funcao.
* Cada funcao de ordenacao precisa de um ponteir para uma funcao de comparacao.
* qsort() ordena uma array.
* Array de ponteiros para funcoes podem te ajudar a executar funcoes diferentes para tipos de dados diferentes.
* Funcoes de comparacao decidem como ordenar dois dados individuais.
* Funcoes com um numero de parametros se chama "variaveis".
* strarg.h possibilita a criacao de funcoess variaveis.
