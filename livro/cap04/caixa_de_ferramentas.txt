CAIXA DE FERRAMENTAS C

Voce ja tem o capitulo 4 no seu repertorio e, agora, acrescentou tipos de dados
e arquivos heard a sua caixa de ferramentas.

* chars sao numeros.
* use shorts para numeros inteiros pequenos
* use ints para a maioria dos numeros inteiros
* use longs para numeros inteiros muito grandes.
* use doubles para numeros de ponto flutuante muito precisos.
* use floats para a maioria dos numeros de ponto flutuante.
* Separe a declaracao de funcao e definicao de funcao.
* Coloque declaracoes em um arquivo header.
* #include<> para headers de biblioteca.
* #include"" para headers locais.
* Salve o codigo objeto em arquivos para agilizar seus builds
* Use make para gerenciar seus builds.
