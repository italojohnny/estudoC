CAIXA DE FERRAMENTAS C

Voce ja tem o capitulo 3 no seu repertorio e, agora, acrescentou o basico do C
a sua caixa de ferramentas.

* Funcoes de C, como printf() e scanf() usam saida padrao e a entrada padrao para se comunicar
* Voce pode mudar onde a entrada, saida e erro padrao sao ligados usando redirecionamento.
* Parametros de linha de comando sao passados para a main() como array de ponteiros string.
* A funcao getopt() facilita a leitura de opcoes de linha de comando.
* A saida padrao vai ao display por default.
* A entrada padrao le a partir do teclado.
* O erro padro e uma saida separada e projetada para mensagens de erro.
* Voce pode escrever no erro padrao, usando fprintf(stderr,...).
* Voce pode criar fluxos de dados customizados com fopen("nomedoarquivo", modo).
* O modo pode ser "w", para escrever; "r", para ler; ou "a" para acrescentar.
