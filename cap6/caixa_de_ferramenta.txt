CAIXA DE FERRAMENTAS C

Voce ja tem o capitulo 6 no seu repertorio e, agora, acrescentou estruturas de
dados e memoria dianamica a caixa de ferramentas.

* Uma lista encadeada e mais extensivel que uma array.
* Dados podem ser unseridos facilmente em uma lista encadeada.
* Uma lista encadeada e uma estrutura de dados dinamica.
* Estrutura de dados dinamica usam struct recursivos.
* Structs recursivos contem uma ou mais conexao para um dado semelhante.
* malloc() aloca memoria no heap.
* free() libera a memoria no heap.
* strdup vai criar uma copia de uma string no heap.
* Um vazamento de memoria e memoria que nao pode mais ser acessada.
* valgrind pode te ajudar a detectar vazamentos de memoria.
